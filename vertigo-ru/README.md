## Краткое содержание

**Поззи** — безжалостный убийца фурри, который ведёт учёт своих жертв в блокноте. После выигрыша загадочного джекпота «DANGER» в казино Лас-Вегаса он оказывается в ловушке 355-этажного здания, полного ужасов. Во время спуска он встречает:

- Мутировавших фурри различных классов
- Воскресающих врагов (Тони Старк, MORGENSTERN)
- Загадочных диспетчеров и вертолёты
- Сюрреалистичную физику и логику снов
- Подземные казино в советском стиле
- Временные петли и параллельные реальности

История сочетает тёмную комедию, экшен и абсурдистскую фантастику с логикой видеоигр (чит-коды, механика возрождения, счётчик убийств).

## Порядок чтения

Читать главы в числовом порядке (ch0 → ch1 → ch2 → ch3), внутри каждой главы — файлы в числовом порядке.

**Полный порядок файлов:**

```
Глава 0 (Пролог):
ch0/tt#0.txt → tt#1.txt → tt#2.txt → tt#3.txt → tt#4.txt → tt#5.txt → 
tt#6.txt → tt#7.txt → tt#8.txt → tt#9.txt → tt#10.txt

Глава 1 (Казино):
ch1/cs#1.txt → cs#2.txt → cs#3.txt → cs#4.txt → cs#5.txt → cs#6.txt → cs#7.txt

Глава 2 (Сон):
ch2/mm#1.txt → mm#2.txt → mm#3.txt → mm#4.txt → mm#5.txt

Глава 3 (Воздушное пространство):
ch3/sp#1.txt → sp#2.txt → sp#3.txt → sp#4.txt → sp#5.txt → sp#6.txt → sp#7.txt → sp#8.txt
```

## Примечания

- История продолжается (в конце каждой главы «Продолжение следует...»)
- Глава 1 заканчивается как «Конец первой части»
- Счётчик убийств достигает **1538** к концу главы 3
- Английский перевод доступен в папке `/vertigo-en/`
